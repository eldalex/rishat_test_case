для самостоятельного развртывания надо пройти по шагам.
1) Создать директорию и перейти в неё.
2) Выполнить: git clone https://github.com/eldalex/rishat_test_case
3) перейти в подкаталог rishat_test_case/rishattest_root
4) ввести команду docker-compose up -d
При развертывании в БД добавляется пользователь admin с паролем admin
а так же добавляются тестовые данные:
три предмета для покупки
два заказа
tax и discount

После запуска контейнера перейти в браузере по адресу:
http://127.0.0.1:8000/admin - админка django. log/pas: admin/admin
http://127.0.0.1:8000/ - стартовая страница с готовым ссылками.
http://127.0.0.1:8000/item/id страница предмета с кнопкой checkout
http://127.0.0.1:8000/buy/id - купить предмет. вернёт id сессии и url для покупки.
http://127.0.0.1:8000/buy/order/id - покупка нескольких товаров в одном заказе.
